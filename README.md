# Code Box

Gerador de imagem de bloco de codigo em PNG transparente, no estilo janela macOS.

Ao vivo: https://paulozeratta.github.io/code-box/

Ferramenta standalone do ecossistema Oceon. Um unico HTML self-contained (fonte embutida, highlight via Prism, render em SVG nativo): cola o codigo, escolhe titulo e linguagem, exporta o PNG com fundo transparente e cantos arredondados.

Suporta Lua, JavaScript, TypeScript, Python, HTML, CSS, JSON, SQL e Bash.

O `index.html` e o build de producao gerado a partir do repositorio privado `oceon-ui-machine` (`src/tools/code-box`).
